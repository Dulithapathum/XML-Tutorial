XPath Expression Details

1. Select the document node

/

This selects the root node of the XML document. It represents the entire XML document as a whole.

2. Select the 'root' element

/movies

This selects the movies element, which is the root element of the document.

3. Select all 'movie' elements that are direct children of the 'movies' element

/movies/movie

This selects all movie elements that are directly under the movies element.

4. Select all 'name' elements that are direct children of the 'movie' element

/movies/movie/name

This selects all name elements that are directly within any movie element.

5. Select all the elements in the document

//*

This selects every element in the entire XML document, regardless of its position.

6. Select all 'name' elements regardless of their positions in the document

//name

This selects all name elements anywhere in the document, no matter where they are located.

7. Select the first 'movie' element

/movies/movie[1]

This selects the first movie element within the movies element.

8. Select all the children of the 'cast' element of the fifth 'movie' element

/movies/movie[5]/cast/*

This selects all child elements of the cast element that is within the fifth movie element.

9. Select the last actor of every 'movie' element

/movies/movie/cast/actor[last()]

This selects the last actor element within each cast element inside all movie elements.

10. Select the first 3 'movie' elements

/movies/movie[position() < 4]

This selects the first three movie elements under the movies element.

11. Select the textual value of the 'imdb' element of all 'movie' elements

//imdb/text()

This selects the text content within all imdb elements across the document.

12. Select the textual value of the second movie 'name' element

/movies/movie[2]/name/text()

This selects the text content within the name element of the second movie element.

13. Select the textual value of the last movie 'name' element

/movies/movie[last()]/name/text()

This selects the text content within the name element of the last movie element.

14. Select all 'movie' elements with the 'release' element value as '2021-10-29'

/movies/movie[release = "2021-10-29"]

This selects all movie elements where the release element has the value 2021-10-29.

15. Select all 'movie' elements with the 'imdb' element value greater than or equal to '7.0'

/movies/movie[imdb >= "7.0"]

This selects all movie elements where the imdb element has a value greater than or equal to 7.0.

16. Select the 'id' attribute of the 'actor' elements that are direct children of the 'cast'

/movies/movie/cast/actor/@id

This selects the id attribute of all actor elements that are directly within any cast element.

17. Select all 'movie' elements that have an 'id' attribute

//movie[@id]

This selects all movie elements that have an id attribute.

18. Select all 'name' elements that have a 'lang' attribute value of 'en'

/movies/movie/name[@lang = "en"]

This selects all name elements under movie elements where the lang attribute is en.

19. Select all 'movie' elements that have a 'lang' attribute value of 'hi'

/movies/movie[name[@lang = "hi"]]

This selects all movie elements that contain a name element with a lang attribute value of hi.

20. Select textual value of the 'name' element of all 'actor' elements that have a 'gender' attribute value of 'female'

//actor/name[@gender = "female"]/text()

This selects the text content of name elements for all actor elements that have a gender attribute set to female.

21. Select all 'boxoffice' elements AND all 'imdb' elements

//boxoffice|//imdb

This selects all boxoffice and imdb elements in the document.

22. Select the name of the second element of the first 'movie' element

name(//movie[1]/*[2])

This returns the name of the second child element of the first movie element.

23. Select the numeric value of the 'imdb' element of the last 'movie' element

number(/movies/movie[last()]/imdb)

This returns the numeric value of the imdb element for the last movie element.

24. Select the string representation value of the 'country' element of the 'movie' element with imdb equal to '4.3'

string(//movie[imdb = "4.3"]/country)

This returns the string value of the country element within the movie element where the imdb element has a value of 4.3.

25. Select the length of the 'name' element's textual value of the first 'actor' element of the sixth 'movie' element

string-length(//movie[6]/cast/actor[1]/name/text())

This returns the length of the text content of the name element for the first actor in the sixth movie element.

26. Select the local name of the 'country' element of the first 'movie' element, i.e., without the namespace

local-name(//movie[1]/country)

This returns the local name (i.e., the name without any namespace prefix) of the country element in the first movie element.

27. Select the number of 'movie' elements

count(/movies/movie)

This returns the number of movie elements within the movies element.

28. Select the sum of all 'age' elements of all 'actor' elements of the second 'movie' element

sum(//movie[2]/cast/actor/age)

This returns the sum of the values of all age elements for all actor elements within the second movie element.
